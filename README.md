# Racinges

## Описание

`Racinges` — компактная 3D-аркада про уличные гонки с быстрым темпом, гаражом, выбором машин и отдельными гоночными сессиями. Игра опирается на атмосферу уличного соперничества и понятную аркадную управляемость: заметное ускорение, читаемый занос, ощутимое нитро и ясная структура прогрессии без открытого мира и без перегруженных систем.

Целевой состав режимов:

- заезды против компьютера с выбором сложности;
- локальный split-screen на двух игроков на одной клавиатуре;
- гараж с выбором машины, просмотром характеристик и визуальным preview;
- понятные гоночные экраны: выбор, гонка, пауза, результаты.

На первом этапе проект собирается вокруг одного тестового кольцевого трека и стартового гаража из пяти машин, чтобы механики, UI и игровой цикл можно было быстро проверять и доводить.

## Запуск

1. Установите `Godot 4.x`.
2. Клонируйте репозиторий и откройте проект через файл `project.godot`.
3. Нажмите `Run Project` в редакторе Godot.

Текущее состояние проекта:

- `Main Menu` с компактным русскоязычным UI, кнопкой `Играть`, переходом в отдельный выбор режима и прямым входом в гараж;
- одиночный поток `Mode Select -> Difficulty Select -> Car Select -> Race -> Results`;
- кооп-поток `Mode Select -> Coop Prep -> Split-Screen Race -> Results` с отдельным окном подготовки, выбором машин для `P1` и `P2` и двумя `HUD`;
- тестовая кольцевая трасса на базе `StoneBrook` с цветными полупрозрачными стартом, финишем и чекпоинтами;
- в текущем MVP трасса не использует жёсткие невидимые стены и не штрафует скоростью за выезд: чекпоинты и прогресс круга продолжают работать;
- заезд против ИИ с тремя сложностями, реальным `HUD`, таймером, нитро на `R`, паузой на `Esc` и экраном результатов;
- локальный кооп теперь запускает двух игроков на одной трассе в split-screen, с коллизиями между машинами;
- `Garage` с набором машин `BMW M3 GTR`, `Dodge Challenger SRT`, `Toyota AE86 Trueno`, `Retro Sprint 80` и `Urban Racer`, где `BMW M3 GTR` в начале отображается как закрытая.

Базовое управление в одиночном режиме:

- `W` — газ;
- `S` — тормоз / реверс;
- `A` / `D` — поворот;
- `R` — нитро;
- `Esc` — пауза.

Базовое управление в локальном коопе:

- `P1`: `W` / `S` / `A` / `D`, нитро — `R`;
- `P2`: стрелки, нитро — `M`;
- в окне подготовки `P1` листает машины через `W` / `S`, `P2` — стрелками вверх / вниз.

## Использованные технологии

- `Godot 4.x`
- `GDScript`
- встроенная 3D-сцена Godot для трека, камер и машин
- встроенная UI-система Godot (`Control`, `CanvasLayer`, темы и стили)
